# Contribuir a Patagonia Browser

¡Gracias por querer colaborar con Patagonia Browser!

## Objetivo

Nuestro objetivo es crear un navegador moderno, rápido, seguro y privado.

Cada cambio deberá mejorar alguno de estos aspectos.

---

## Principios

- El rendimiento es prioritario.
- La privacidad nunca será opcional.
- El código debe ser limpio y fácil de mantener.
- Las nuevas funciones deberán estar documentadas.
- Ningún cambio deberá degradar la experiencia del usuario.

---

## Estilo

- Código simple.
- Comentarios claros.
- Nombres descriptivos.
- Evitar código duplicado.

---

## Filosofía

Patagonia Browser busca ofrecer una experiencia moderna, elegante y segura para cualquier usuario.
